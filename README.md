Airbnb Clone

This is a full-stack Airbnb clone web application built using Node.js, Express, EJS, Passport.js, and Bootstrap. The project replicates core features of Airbnb like user authentication, property listings, and basic CRUD operations.

Features

User Authentication (Login/Register/Logout using Passport.js)
Create, Edit, and Delete Listings
Upload and manage property images
Add property details (title, description, price, location)
Flash messages for better user interaction
Responsive UI using Bootstrap
Secure routes and session handling

Tech Stack

Backend:Node.js, Express.js  
Frontend: EJS, CSS, Bootstrap, JavaScript  
Authentication: Passport.js  
Database: MongoDB  
Version Control: Git & GitHub  



Project Structure
├── models  Database models
├── routes  Express routes
├── views  EJS templates
├── public  Static files (CSS, JS, images)
├── middleware  Custom middleware
├── app.js  Main server file



 Installation & Setup

1. Clone the repository:

  git clone https://github.com/sofifaizzan/airbnb.git

2. Navigate to the project folder:
  cd airbnb

3. Install dependencies:
  npm install

4. Create a .env file and add:
  SESSION_SECRET=your_secret_key
  DB_URL=your_mongodb_connection

5. Start the server:
  npm start

6. Open server
   http://localhost:3000
